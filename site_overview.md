## 🎨 Design System: Color Palette
### 🌌 Core Website Colors
These colors form the foundation of the dark mode system used throughout the main website.
| Color Variable | CSS Custom Property | OKLCH Value | HEX Value | Description & Usage |
| :--- | :--- | :--- | :--- | :--- |
| **Deep Background** | `--bg-deep` | `oklch(0.12 0.006 260)` | `#050608` | Global page background, HTML, and base body element. |
| **Base Background** | `--bg-base` | `oklch(0.16 0.008 260)` | `#0b0d11` | Standard content blocks, sections, and container surfaces. |
| **Raised Background** | `--bg-raised` | `oklch(0.20 0.01 260)` | `#13161b` | Cards, elevated navigation components, and floating UI surfaces. |
| **Primary Text** | `--text-primary` | `oklch(0.93 0.006 260)` | `#e5e8ec` | Main headings, body text, primary button labels, and high-emphasis text. |
| **Secondary Text** | `--text-secondary` | `oklch(0.78 0.008 260)` | `#b4b7bd` | Subheadings, standard paragraph copy, and medium-emphasis text. |
| **Muted Text** | `--text-muted` | `oklch(0.70 0.006 260)` | `#9c9ea2` | Form captions, small labels, footer text, and disabled states. |
| **Accent Blue** | `--accent` | `oklch(0.72 0.16 264)` | `#70a1ff` | Brand accent color, primary links, action buttons, and active indicator highlights. |
| **Accent Hover** | `--accent-hover` | `oklch(0.78 0.18 264)` | `#7bb2ff` | Hover and focus state tint for brand accent items. |
| **Border Color** | `--border` | `oklch(0.24 0.008 260)` | `#1d1f23` | Divider lines, container outlines, and input border frames. |
### 🚦 System & Status Colors
Used to denote operational health, statuses, and live metrics in charts and data tables.
| Status | CSS Custom Property | OKLCH Value | HEX Value | Description & Usage |
| :--- | :--- | :--- | :--- | :--- |
| **Idle** | `--status-idle` | `oklch(0.50 0.00 0)` | `#636363` | Off, disconnected, or inactive device states. |
| **Active / Green** | `--status-active` | `oklch(0.65 0.15 150)` | `#3aa85b` | Healthy, online, or running operations. |
| **Error / Red** | `--status-error` | `oklch(0.60 0.18 25)` | `#d74745` | Warnings, system faults, or disconnected nodes. |
### 📊 LaundryLink Dashboard Mockup Colors
These colors are specifically used within the interactive dashboard mockup component to simulate a professional enterprise UI layout.
*   **Dashboard White**: `#f7f8fa` (`oklch(0.98 0.003 260)`) — White background panels and card backdrops.
*   **Dashboard Accent Surface**: `#131a29` (`oklch(0.22 0.03 264)`) — Deep charcoal-blue accent surface.
*   **Mockup App Text (Slate)**: `#0f172a` — High-contrast dark gray text for main mockup data tables.
*   **Mockup Secondary Text**: `#334155` — Secondary text inside mockup headers.
*   **Mockup Border**: `#cbd5e1` / `#e2e8f0` — Slate dividers and borders separating columns.
*   **Mockup Card Gradient**: Multi-stop gradient used inside mockup metric highlights to suggest depth.
    *   **Start**: `#0c1936` (`oklch(0.22 0.06 264)`)
    *   **Middle**: `#0e0f23` (`oklch(0.18 0.04 280)`)
    *   **End**: `#050b18` (`oklch(0.15 0.03 264)`)
